# Security Policy

Security vulnerabilities should be disclosed to the project maintainers through [Immunefi], or alternatively by email to security@pieces.market.

[Immunefi]: https://immunefi.com

## Bug Bounty

Responsible disclosure of security vulnerabilities is rewarded through a bug bounty program on [Immunefi].

There is a bonus reward for issues introduced in release candidates that are reported before making it into a stable release. 

## Security Patches

Security vulnerabilities will be patched as soon as responsibly possible.

### Supported Versions

Security patches will be released for the latest minor of a given major release. 

Note as well that the Solidity language itself only guarantees security updates for the latest release.
